PrintUpgraded Filament Temperature Cheat Sheet

Quick-reference print settings for the 6 most common 3D printing filaments

Beginner-Friendly Intermediate Advanced
Filament Nozzle Temp Bed Temp Print Speed Cooling Fan Tips
PLA 190 - 220 C 50 - 60 C 40 - 60 mm/s 100% Easiest material. Start here. Great for decorative prints and prototypes.
PETG 220 - 250 C 70 - 85 C 30 - 50 mm/s 30 - 50% Reduce retraction to fight stringing. Use Z-hop. Don't squish first layer too much.
ABS 230 - 260 C 95 - 110 C 40 - 60 mm/s 0% Needs enclosure. Ventilate for fumes. Prone to warping without stable temps.
TPU (95A) 210 - 230 C 40 - 60 C 15 - 25 mm/s 50% Direct drive recommended. Disable retraction if Bowden. Print slow.
ASA 235 - 260 C 90 - 110 C 40 - 50 mm/s 0 - 30% Like ABS but UV-resistant. Needs enclosure. Great for outdoor parts.
Nylon 240 - 270 C 70 - 90 C 30 - 50 mm/s 0% Dry thoroughly before printing. Very hygroscopic. Use a dry box while printing.

Before Every Print

  • Check filament spool for moisture damage
  • Clean nozzle with a cold pull if switching materials
  • Verify bed is level and clean
  • Confirm slicer profile matches your filament

Tuning Tips

  • Start at the low end of the temp range
  • Raise temp 5 C at a time if layers look weak
  • Lower temp 5 C at a time if you see stringing
  • Always print a temp tower with a new brand

Storage Reminders

  • PLA: store sealed with desiccant
  • PETG: moderate moisture sensitivity
  • Nylon/TPU: dry box required
  • ABS/ASA: less sensitive, but seal anyway
